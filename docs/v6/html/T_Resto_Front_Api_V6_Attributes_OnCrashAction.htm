Default0 Default action will be applied based on internal iikoFront app logic.
Restart1 iikoFront app will restart the plugin if it crashes. It may be useful as a temporary solution for plugins having rare harmless bugs, which terminate a process and cannot be fixed immediately. Anyway, bugs must be fixed sooner or later, exceptions shouldn't occur or have to be gracefully handled.
DoNotRestart2 iikoFront app won't restart the plugin if it crashes.
